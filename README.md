# rr_blue_green_LED
Contains arduino code and ROS package for LED control with MultiMoto hat

LEDs:
  VCC PCL22 AC/DC 12V
  1 x blue, 1 x green

MultiMoto hat:
  Robot Power MultiMoto
  (4 channels, 6 to 36 V battery)
  Info:
    http://www.robotpower.com/products/MultiMoto_info.html
    
Arduino:
  Arduino UNO 
  T.E. 011 94V-0
